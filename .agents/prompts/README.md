# Meta-prompts

Project-specific prompt templates and style guides for AI-assisted tools (e.g. image generation, code gen).

- **`image-style.md`** – Default aesthetic for generated images (marker-on-whiteboard). Reference this when using nano-banana or similar so blog visuals stay consistent.

Add more files here for other meta-prompts (e.g. `blog-voice.md`, `diagram-conventions.md`).
