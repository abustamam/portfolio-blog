# Image generation style (meta-prompt)

Use this when generating images for the blog (e.g. with nano-banana or other image tools).

## Default aesthetic: marker-on-whiteboard, diagram-first

- **Style:** Hand-drawn marker on whiteboard
- **Look:** Clean white background, bold marker lines (dry-erase style), technical/diagram feel
- **Composition:** Lean into diagrams. Keep it concise: title + diagram only. Minimal text, no extra labels or clutter.

When asking for an image, prepend or include:

> Style: marker on whiteboard, hand-drawn diagram aesthetic, white background, bold marker lines. Concise: title + diagram only, minimal text, no labels.

Adjust the diagram type per image (e.g. "simple flowchart", "system boundary diagram", "connected nodes") but keep the marker-on-whiteboard look and minimal composition consistent.
