# Image generation style (meta-prompt)

Use this when generating images for the blog (e.g. with nano-banana or other image tools).

## Default aesthetic: marker-on-whiteboard

- **Style:** Hand-drawn marker on whiteboard
- **Look:** Clean white background, bold marker lines (dry-erase style), technical/diagram feel
- **Use for:** Diagrams, architecture sketches, concept illustrations, hero images for posts

When asking for an image, prepend or include:

> Style: marker on whiteboard, hand-drawn diagram aesthetic, white background, bold marker lines.

Adjust per image (e.g. "simple flowchart", "system boundary diagram") but keep the marker-on-whiteboard look consistent across the blog.
